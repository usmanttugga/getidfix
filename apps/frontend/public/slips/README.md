# Slip Sample Images

Place the slip sample images in this folder with the following exact filenames:

- `basic-slip.png` — Basic Slip (Verified NIN Details)
- `premium-slip.png` — Premium Slip (Digital NIN Slip)
- `regular-slip.png` — Regular Slip (NIN Slip / NINS)
- `vnin-slip.png` — VNIN Slip (Verification-as-a-Service)
- `standard-slip.png` — Standard Slip (NIN Card Plain)
